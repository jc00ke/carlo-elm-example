# Carlo + Elm

This is an example [Elm](https://elm-lang.org/) application using
the [Carlo](https://github.com/GoogleChromeLabs/carlo) app framework.

## Setup

* Install `node >= v7.6.0`
* Install `Google Chrome/Chromium >= 70`
* Install `elm >= 0.19.0`

```
> npm i
> elm make src/App.elm --output=app.js
> npm start
```
